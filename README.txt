🪄 Yugioh AutoCraft - Hướng dẫn sử dụng

🌟 Yêu cầu trước khi chạy:
✅ Bắt buộc chạy file `yugiohautocraft.exe` dưới quyền **Administrator (Run as Administrator)**.
✅ Vào game -> Submenu -> Game Settings -> General:
    - Đặt Resolution: **1600x900**
    - Đặt View mode: **Window mode**
✅ Vào deck và để nguyên phần deck bình thường.

---

⚙ Cách sử dụng:
1️⃣ Chạy file `yugiohautocraft.exe`  
2️⃣ Trong giao diện, bạn có thể:
   - Nhập trực tiếp tên lá bài muốn craft vào ô trống
   - Hoặc copy/paste từ nơi khác vào ô nhập
3️⃣ Điều chỉnh số lượng muốn craft bằng nút [+] hoặc [-]
4️⃣ Nhấn **Run Auto Generate**  
✨ Chương trình sẽ tự động thao tác trong game để craft đúng số lượng lá bài.

---

📌 Lưu ý:
- Giữ cửa sổ game luôn hiện trên màn hình.
- Nếu craft nhiều lá, chương trình sẽ tự động chạy lần lượt.

🪄 Yugioh AutoCraft - Quick Start Guide (English)

🌟 Requirements before starting:
✅ MUST run `yugiohautocraft.exe` as **Administrator (Right click → Run as Administrator)**.
✅ In game → Submenu → Game Settings → General:
    - Set Resolution: **1600x900**
    - Set View mode: **Window mode**
✅ Go into your deck screen and leave it open as usual.

---

⚙ How to use:
1️⃣ Run `yugiohautocraft.exe`  
2️⃣ In the app UI, you can:
   - Type the card name directly
   - Or copy/paste from somewhere else
3️⃣ Adjust the quantity to craft using [+] or [-] buttons
4️⃣ Click **Run Auto Generate**  
✨ The program will automatically perform all steps to craft the selected cards.

---

📌 Note:
- Keep the game window visible on your screen.
- If crafting multiple cards, the program will process them one by one.
